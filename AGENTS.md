# 《上海夜未央》AGENTS.md

## Project Overview
This project is a cinematic music-video and story-world project set in 1920s–1930s Shanghai.

## Core Direction
- Focus on 1930s Shanghai show business, film studios, singers, dancers, jazz clubs, romance, and city atmosphere.
- Do not make bloody mafia/action stories the main direction.
- Underworld elements may appear only as subtle atmosphere or hints.
- Songs should be treated as theme songs for fictional films, fictional revues, and fictional nightclubs.

## Visual Rules
- Main aspect ratio: vertical 9:16, 1080×1920 proportion.
- Avoid modern Shanghai elements:
  - no skyscrapers
  - no Oriental Pearl Tower
  - no modern cars
  - no modern neon signs
  - no contemporary clothing
- Use 1920s–1930s Shanghai atmosphere:
  - French Concession edge
  - Bund / Huangpu River
  - Shikumen alleys
  - wet stone roads
  - old European-style buildings
  - warm street lamps
  - cool blue rainy shadows

## Music Direction
- Do not make every song generic “Shanghai jazz.”
- Use varied influences:
  - French chanson
  - Paris musette
  - New York swing
  - Broadway revue
  - Charleston
  - tango
  - bolero
  - early Showa kayokyoku
  - vintage film orchestra
- Add “1930s Shanghai” as flavor, not as the only style.
